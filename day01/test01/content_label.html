万维网联盟(World Web Consortium)
W3C专门为了定义网页相关的标准而建立
W3C定义了网页中的HTML CSS Dom HTTP XML 等标准
HTML(Hyptertext Markup Language)是超文本标准语言，它负责网页的三个要素之中的结构
HTML是一种规范，一种标准，它通过标记符号来标记要显示的网页中的而各个部分。网页文件本身是一种文本文件中添加标记符，可以告诉浏览器如何显示其中的内容（如文字如何处理、画面如何安排、图片如何显示等） 浏览器安顺序阅读网页文件然后根据标记符解释和显示其标记的内容，对书写错的标记将不指出其错误。且不停止其解释执行过程，编织着只能通过显示效果来分析出错原因。 标签
1. 标签成对出现，少数单个
2. HTML标记指标签
3. 完整的标签称为元素
4. <p>段落<em>段落子标签
5. <hr>下划线
6. <br>换行符
属性
提供附加信息
Id是标签的唯一标识，不能出现相同的id属性值
Class属性为标签分组，拥有相同class属性的标签是一组可以出现相同的class属性，可以为一个元素指定多个class
Title属性指定标签的标题，指定title之后鼠标移动到元素上方式，会出现提示文字。
